**This is a User Authenticated Bloging web app.
User First need to go to http://localhost:5173/login to login first ot If new user then click the signUp link below Login form

Then It will redirect you to the home page :- http://localhost:5173/
where:-
1. user can logout
2. User can Add new Blog by clicking the Add-blog Button
3. User can view each Blog Content by clicking on each blog

4. User can upload a blog with some CoverImage which will be displayed on each blog content
5. The backned used MongoDB+Express.js+Cloudinary+Multer for smooth Backend service
6. Tested each API in (Postman) and take care of all eadge cases in " Form validation "
7. Frontend used ;- React.js + ShadCn Ui + Tailwind CSS and for Fetching APIs like All blogs Blogs by Id,
8. created custom hooks using React Query or TanStack Query.

To Open this app:-
1. Go to http://localhost:5173/login
2. then http://localhost:5173/
3. By clicking any blog you open :- http://localhost:5173/blog/:id


NOTE : I have not implented the "routes protection"
If given extra time I would use "Redux tool kit" for better State management & implementing more functionality !!
